# 🛒 Superstore Sales Analysis
### Using SQL Server & Power BI

> This project presents an end-to-end data analysis of the Superstore dataset. 
> Starting with raw data in SQL Server, the data was cleaned, duplicates were removed, 
> and structured views were built. The final results were visualized through an 
> interactive Power BI dashboard covering sales performance, customer behavior, 
> and product insights.

---

## 🛠️ Tools Used
- **SQL Server** — Data cleaning, EDA, Views
- **Power BI** — Interactive Dashboard

---

## 🔄 Project Steps
1. Created Database & imported raw data
2. Exploratory Data Analysis (EDA)
3. Checked & handled NULL values
4. Removed duplicates using CTE + ROW_NUMBER()
5. Created SQL Views following Star Schema
6. Built Power BI Dashboard with 3 pages

---

## 📊 Dashboard Pages
- **Overview** — KPIs + Sales by Year, Month & Category
- **Sales** — Sales by Region, Sub-Category, State & Segment
- **Customer & Product** — Top Customers, Categories & Sub-Categories

---

## 📈 Key Insights
- Total Sales: **$2.26M**
- Total Orders: **4,922**
- Total Customers: **793**
- Best Year: **2018 — $721K**
- Top Region: **West**
